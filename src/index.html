 Focus XP - a pomodoro timer that looks like Windows XP
 Built to help you get stuff done without distractions.

 How Focus XP works:
    - Set a task in the sidebar
    - Start and work until the timer ends
    - Take a break
    - Repeat
    - Check your weekly stats to see how youre doing

 Keyboard shortcuts:
   SPACE  .....  start / pause
   R      .....  reset timer
   ESC    .....  close popup

 Your data (tasks, sessions, weekly stats) is saved locally in your browser; no account needed.

 Thank you for checking out Focus XP!
